# consumer-study

An open-source skill for AI coding agents that runs the complete consumer research pipeline: from research question to published study with a DOI.

## What it does

You describe what you want to test with real consumers. The skill handles everything:
survey design, participant recruitment, statistical analysis, and academic reporting.

## Validated

Built from the Gateway Study methodology. Tested with 1,027 real participants.
Published results: https://doi.org/10.17605/OSF.IO/73NJW

## Requirements

- An AI coding agent (Claude Code, Copilot CLI, Codex, Gemini CLI)
- API keys: Typeform, Prolific, Anthropic
- Optional: Vercel (deployment), OSF (data deposit)

## Quick start

Install the skill in your agent, then say:
"I want to test whether [X] changes how consumers think about [Y]"

The skill takes it from there.
